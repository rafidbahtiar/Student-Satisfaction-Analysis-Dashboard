# Student Satisfaction Analysis Dashboard
This project analyzes student satisfaction across various academic, financial, and demographic factors using Power BI. The goal is to identify key drivers of student satisfaction and provide actionable insights for improving the overall student experience in higher education institutions.

## Objectives
- Understand overall student satisfaction levels.
- Analyze the impact of teaching quality and campus facilities on satisfaction.
-	Evaluate how financial factors such as tuition and scholarships affect student experience.
-	Explore satisfaction differences across demographic groups.
